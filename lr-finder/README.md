# Tensorflow 2 learning rate finder

A Tensorflow 2 compatible learning rate finder. See https://www.avanwyk.com/finding-a-learning-rate-in-tensorflow-2/ for
more details.

`lr_finder.py` contains a standalone implementation using exponential annealing and a loss smoothing.

A complete example is given in the `tensorflow2_lr_finder.ipynb` notebook.

Implementation requires tensorflow==2.0.0+ installed.
